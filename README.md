# Workout-Analytics-Project

This project tracks and analyzes personal workout data using Excel, SQL, and Power BI.

## Project Goals
- Track workout frequency
- Measure strength progress
- Analyze workout duration trends

## Tools
- Excel
- SQL
- Power BI
- GitHub
